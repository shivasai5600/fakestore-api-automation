# FakeStore API Automation (Python + Pytest)

## Overview
This project demonstrates API automation using Python, Pytest, and Requests.

## Features
- Modular framework design
- Logging implemented
- HTML reporting
- CI/CD using GitHub Actions

## Test Coverage
- Get all products
- Get single product
- Create product
- Get categories
- Filter by category
- Negative testing
